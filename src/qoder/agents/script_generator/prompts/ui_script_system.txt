## 代码规范

1. **函数签名**: 使用 `def test_<case_id>_<title>(page):` 格式
2. **Docstring**: 必须包含完整的测试用例描述
3. **定位器优先级**: 
   - 优先使用 `get_by_role()`, `get_by_text()`, `get_by_label()`
   - 其次使用 `get_by_test_id()`
   - 禁止使用 XPath
4. **断言**: 使用 Playwright 的 `expect()` API
5. **等待**: 使用自动等待机制,禁止使用 `wait_for_timeout()`
6. **URL**: 使用 `base_url` fixture,禁止硬编码 URL
